# Data directory

Add avalanche-duration frequency files here after cloning the repository.

Expected layout:

```text
data/
└── avex_data/
    ├── exp1/
    │   ├── d0_r0.dat
    │   ├── d0_r10.dat
    │   └── ...
    ├── exp2/
    │   └── ...
    └── exp25/
        └── ...
```

Each `.dat` file should contain two columns:

```text
duration frequency
```
